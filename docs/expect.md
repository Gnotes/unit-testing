# Expect

**Expect** 有两个包 [`expect`](https://github.com/mjackson/expect) 和 [`expect.js`](https://github.com/Automattic/expect.js)

- expect: 在 `v21+` 以后已经合并到 `Jest/expect` 中，因此使用 Jest 吧
- expect.js: 是基于 `Should` 的 BDD 断言，与 Should 差别不大，那咱使用 Should 吧
